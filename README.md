# Z-izol s.r.o.

Webová stránka stavebnej spoločnosti Z-izol s.r.o.

## Nasadenie

Stránka sa automaticky nasadzuje na GitHub Pages cez GitHub Actions pri každom pushi do vetvy `main`.

### Prvotné nastavenie repozitára

1. V repozitári choď na **Settings → Pages**
2. V sekcii **Build and deployment** nastav **Source: GitHub Actions**
3. Push do `main` spustí deploy automaticky

URL stránky nájdeš v Settings → Pages po prvom úspešnom nasadení.
